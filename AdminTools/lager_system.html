→
Gesamter Ablauf ungetestet — die Kette Inventur → Auswertung → WhatsApp → Import → Analyse wurde noch nicht komplett durchgespielt.
→
BOS-SNAP Format — das Format des versteckten Codes in der WhatsApp-Nachricht muss beim ersten echten Test geprüft werden.
→
baeko_bestand.js fehlt oder leer — alle Tools die darauf aufbauen zeigen Fehler bis die Stammdaten gepflegt sind.
→
Analyse braucht Daten — baeko_lager_analyse macht erst Sinn wenn mindestens 4–6 Wochen Bestellhistorie drin sind.
